## 📖 Sobre o Projeto

O **Primeiros Cliques** é uma plataforma web educacional voltada para a **inclusão digital no Brasil**, desenvolvida como projeto acadêmico no primeiro período do curso de Ciência da Computação.

A proposta é simples e poderosa: levar conhecimento digital de forma acessível, clara e progressiva para pessoas que nunca tiveram contato com a tecnologia — idosos, adultos em requalificação profissional, e comunidades com baixo acesso à educação tecnológica.

> *"Acreditamos que a inclusão digital é fundamental para o empoderamento individual e para a construção de uma sociedade mais conectada e informada."*

---

## ✨ Funcionalidades

- 🗺️ **Trilhas temáticas** organizadas por nível de complexidade
- 📚 **Aulas passo a passo** com linguagem simples e direta
- 💡 **Dicas práticas** ao final de cada lição
- 📝 **Exercícios de fixação** com respostas explicadas
- 📱 **Conteúdo voltado para o uso do celular**, o dispositivo mais acessível ao público-alvo

---

## 🗺️ Trilhas de Aprendizado

O site é organizado em **6 trilhas progressivas**, cada uma contendo múltiplas aulas:

| # | Trilha | Descrição |
|---|--------|-----------|
| 1 | 🚀 **Primeiros Passos** | Wi-Fi, navegador, uso de aplicativos — o começo de tudo |
| 2 | 💬 **Comunicação** | WhatsApp: mensagem de texto, áudio e videochamada |
| 3 | 💰 **Finanças Digitais** | App do banco, PIX com segurança, golpes pelo WhatsApp |
| 4 | 🛡️ **Segurança Digital** | Senhas seguras, fake news, como evitar golpes |
| 5 | 🏛️ **Serviços Públicos** | Gov.br, agendamentos online, INSS, Bolsa Família |
| 6 | 💼 **Mercado de Trabalho** | Currículo simples, envio por e-mail, LinkedIn |

## 🎨 Tecnologias Utilizadas

Este projeto foi desenvolvido **100% com tecnologias front-end puras**, sem frameworks ou dependências externas:

- **HTML5** — Estrutura semântica com uso de `<nav>`, `<main>`, `<section>`, `<footer>`
- **CSS3** — Flexbox, variáveis CSS, design responsivo, estilização por página
- **Google Fonts** — Tipografia com a família Poppins
- **Sem JavaScript** — Toda a navegação é baseada em HTML estático

## 📚 Contexto Acadêmico

| Campo | Detalhe |
|-------|---------|
| 🏫 Instituição | Universidade (Curso de Ciência da Computação) |
| 📅 Período letivo | 1º Período — 2026 |
| 📦 Disciplina | Desenvolvimento Front-end |
| 👥 Tipo | Projeto em grupo |
| 🎯 Apresentação | 1ª Avaliação (P1) |

O projeto nasceu de uma discussão sobre **desigualdade digital no Brasil**. O grupo escolheu trabalhar com um público muitas vezes ignorado: pessoas com pouco ou nenhum contato com tecnologia. Cada aula foi pensada com linguagem acessível, exemplos do cotidiano e exercícios simples.

---

## 🧠 Aprendizados Técnicos

Durante o desenvolvimento, foram aplicados os seguintes conceitos de front-end:

- Estruturação de páginas com **HTML semântico**
- Uso de **Flexbox** para layouts responsivos
- Organização de projetos com **múltiplos arquivos CSS**
- Navegação entre páginas com **links relativos**
- Uso de **imagens e assets** externos vinculados corretamente
- Aplicação de **boas práticas de acessibilidade** (textos alternativos, hierarquia de títulos)
